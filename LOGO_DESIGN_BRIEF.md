# CampusVani — Logo Design Brief

Use this document to generate a professional logo for the **CampusVani** platform.

---

## 🏷 Brand Name
**CampusVani**
- "Campus" = University / College life
- "Vani" = Sanskrit for **Voice / Speech**
- Combined meaning: *The Voice of the Campus*

---

## 🎯 Brand Identity & Mood

| Attribute     | Description |
|---------------|-------------|
| **Purpose**   | Anonymous student feedback platform for MANIT Bhopal |
| **Tone**      | Trustworthy, modern, empowering, intelligent |
| **Feeling**   | Safe, secure, futuristic, yet approachable |
| **Keywords**  | Anonymity · AI · Student Voice · Transparency · University |

---

## 🎨 Brand Color Palette

### Primary (Brand) — Indigo/Violet
These are the **dominant colors** — use them for the logo mark and wordmark.

| Token         | Hex       | Usage |
|---------------|-----------|-------|
| `brand-600`   | `#4F46E5` | Primary CTA, logo fill, main color |
| `brand-500`   | `#6366F1` | Highlights, glows |
| `brand-400`   | `#818CF8` | Light mode accents, gradient end |
| `brand-300`   | `#A5B4FC` | Subtle backgrounds, dark mode text |

### Gradient Direction
The logo should ideally use a **left-to-right gradient**:
```
from: #4F46E5  (Indigo 600)
to:   #6366F1  (Indigo 500 / slightly violet)
```
The hero headline in the app uses:
```
from-brand-600 (#4F46E5) → to-indigo-600 (#4338CA)  [light mode]
from-brand-400 (#818CF8) → to-indigo-400 (#818CF8)  [dark mode]
```

### Background Contexts
- **Light mode background**: `#F8FAFF` (near-white with a blue tint)
- **Dark mode background**: `#0D0F1A` (very deep dark blue-black)
- The logo must look **great on both white and very dark backgrounds**.

---

## 🔤 Typography

| Property   | Value |
|------------|-------|
| **Font**   | **Inter** (Google Fonts) |
| **Weight for wordmark** | `800` Extra Bold or `700` Bold |
| **Tracking** | Tight (`tracking-tight`) |
| **Style**  | Modern, clean, geometric sans-serif |

**Font Stack**: `Inter, -apple-system, BlinkMacSystemFont, Segoe UI, Roboto, Arial, sans-serif`

---

## 🖼 Logo Mark Concept (Icon)

The icon/symbol should communicate: **Voice + Shield + Intelligence**

### Option A — Preferred: Speech Bubble + Shield
- A **speech bubble** with a **subtle shield** or lock icon inside it.
- Suggests: *Speak freely, stay protected.*
- The bubble could have a small **AI sparkle** (⚡ or ✦) inside to hint at AI moderation.

### Option B: Microphone + Shield
- A minimal **microphone silhouette** enclosed within a rounded shield.
- Represents: *Your voice is protected.*

### Option C: Abstract Wave
- An abstract sound wave or voice waveform in a circular container.
- Clean and modern, represents "Vani" (voice/sound).

---

## 📐 Logo Composition

### Preferred Layout: **Horizontal (Wordmark + Icon)**
```
[Icon]  CampusVani
```
- Icon on the left, brand name on the right.
- The "C" or "CV" monogram can also be used as a standalone app icon.

### Alternate: **Stacked (Vertical)**
```
  [Icon]
CampusVani
```

---

## ✨ Visual Style

| Property       | Direction |
|----------------|-----------|
| **Style**      | Modern, minimal, slightly futuristic |
| **Finish**     | Flat or subtle gradient (avoid heavy 3D) |
| **Corners**    | Rounded (the app uses `rounded-2xl` = 1rem, `rounded-3xl` = 1.5rem) |
| **Effects**    | Subtle glow / shadow allowed: `0 4px 14px rgba(79,70,229,0.25)` |
| **Complexity** | Simple and recognizable at small sizes (works as a favicon too) |

---

## 🚫 What to Avoid

- ❌ No bright/neon colors (red, green, orange as primary)
- ❌ No overly complex illustrations
- ❌ No serif fonts
- ❌ No generic graduation cap (too cliché)
- ❌ No gradients that are too rainbow-like
